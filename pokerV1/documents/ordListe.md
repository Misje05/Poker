# Ordliste for Poker P20

* **The Flop:** De tre første felleskortene som legges på bordet.
* **Hole Cards:** De to private kortene hver spiller får utdelt.
* **Showdown:** Når gjenværende spillere viser kortene sine for å kåre vinneren.
* **DAO (Data Access Object):** Designmønsteret vi bruker for å snakke med databasen.
* **LCA:** Lifecycle Architecture - vår nåværende milepæl.
