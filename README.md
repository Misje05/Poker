# Poker (Java Web App)

A multiplayer poker game developed as a school project using Java.  
The application supports online tables and multiple players connecting from different devices.
<br>
Want to try it out? 
**Live:** [here](http://ider-database.westeurope.cloudapp.azure.com:8081/poker/login)

---

## Tech Stack

- Java
- Spring MVC
- JSP
- Apache TomEE
- Bonobo Git

---

## Overview

The application is built as a web-based system where:

- Players join poker tables through a browser  
- The server handles game logic and player actions  
- Multiple users can connect simultaneously over a network  

---

## Running the Project

1. Clone the repository from Bonobo Git  
2. Deploy the project to Apache TomEE  
3. Start the server  
4. Open the application in a browser  

---

## Purpose

Built to demonstrate:

- Web development with Spring MVC  
- Server-side rendering with JSP  
- Multiplayer logic and state handling
